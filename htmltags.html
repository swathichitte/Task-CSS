HTML Tags and Their Descriptions

Basic Tags

<a> - Creates a hyperlink to another page or location.

<abbr> - Represents an abbreviation with optional title info.

<audio> - Embeds audio content.

<b> - Makes text bold.

<blockquote> - Indicates a long quotation.

<br> - Inserts a line break.

<button> - Represents a clickable button.

<caption> - Adds a caption to a table.

Structure & Grouping

<code> - Displays code snippets.

<col>, <colgroup> - Define column layout in a table.

<dd> - Description for a term in a definition list.

<del> - Marks text as deleted.

<details> - Creates a collapsible section.

<div> - Container used for grouping content.

<dl>, <dt> - Create definition list and define terms.

<fieldset> - Groups form elements.

<form> - Container for user input.

Layout & Meta

<header> - Introduces a section or page.

<hr> - Thematic break (horizontal line).

<i> - Italic text.

<iframe> - Inline frame to embed other content.

<label> - Caption for form inputs.

<legend> - Caption for a fieldset.

<li> - List item.

<map> - Defines image map regions.

<mark> - Highlights text.

Lists & Inputs

<ol>, <ul> - Ordered and unordered lists.

<option> - Item in a dropdown list.

<p> - Paragraph of text.

<pre> - Preserves formatting and spacing.

<q> - Short inline quote.

<section> - Groups related content.

<select> - Dropdown menu.

<small> - Smaller text.

<span> - Inline container for styling.

<strong> - Important or emphasized text.

<style> - Internal CSS rules.

<sub> - Subscript text.

<sup> - Superscript text.

Tables & Media

<table> - Creates a table.

<thead>, <tbody>, <tfoot> - Table structure.

<td>, <th>, <tr> - Table data, headers, rows.

<template> - Hidden content for JS rendering.

<textarea> - Multi-line text input.

<time> - Time/date value.

<u> - Underlined text.

<video> - Embeds video content.
